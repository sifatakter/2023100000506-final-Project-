## 📋 Project Overview

This repository contains the final project submission for the **Applied Penetration Testing Engineer** course (CADS012). All testing activities were performed exclusively within the assigned TryHackMe rooms in full compliance with the scope of authorization.

## ⚙️ Methodology

This assessment follows a structured penetration testing methodology:

1. **Reconnaissance** — Passive and active information gathering using tools such as Nmap, Gobuster, and WHOIS lookups to enumerate the target surface.
2. **Vulnerability Identification** — Systematic identification of misconfigurations, outdated services, and web application flaws aligned with the OWASP Top 10.
3. **Exploitation** — Controlled exploitation within the authorized lab environment to demonstrate the real-world impact of identified vulnerabilities.
4. **Reporting** — Documentation of findings with security impact analysis and prioritized remediation recommendations.
